# Back-end Practice

## 펫진진 상품 관련 프로그램
### 프로그램 설명: 펫진진 쇼핑몰 상품에 관한 상품 및 재고관리 프로그램
 - 전체 상품 조회
 - 펫 종류별 상품 조회
 - 신규 상품 등록 및 삭제
 - 입고 내역에 따른 재고수량 자동 업데이트
 - 상품 수량 변경

## 사용기술
- java
- mysql
- jdbc
- productdao
- stockdao
